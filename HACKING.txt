Basic get-you-up-and-running instructions.
(Temporary version for Rails 3 beta port)

REQUIREMENTS

 * Ruby >=1.8.7
 * RubyGems >=1.3.5
 * bundler gem >= 0.9
 * MySQL
 ** mysql_config (as e.g. provided by libmysqlclient-dev on Debian)

on Debian:

 * ruby1.8-dev (to get mkmf.rb, which is not in the ruby1.8 package)
 * libopenssl-ruby1.8

INSTRUCTIONS

1. Grab the source:

$ git clone --branch rails3 git://github.com/chrismear/one-click-orgs.git
$ cd one-click-orgs

3. Install the required gems using Bundler:

$ bundle install

3. Set up a MySQL database for development, with settings as in config/database.yml. E.g.:

$ mysql -u root -p
mysql> create database one_click_development;
mysql> create database one_click_test;
mysql> exit;

4. Migrate the database:

$ rake db:setup

5. Start the server:

$ rails server

6. Visit http://localhost:3000/
